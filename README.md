# 🚀 HiBinds Launcher

**HiBinds** is a universal session launcher written in Python. It allows users to create customized macro-binds that launch multiple programs, files, and browser tabs sequentially with precise time delays. It's an ideal tool for gamers (e.g., launching FACEIT Anti-Cheat, Steam, and specific web tabs in one click) and professionals setting up workspaces.

---

## ✨ Features

- **Automated Local Database**: Automatically checks for and initializes a standalone `HIBinds.json` storage file on startup.
- **Dynamic Bind Creation**: Add an infinite sequence of applications (`.exe`) or URLs with tailored delays (in milliseconds) between each action.
- **Fault-Tolerant Runner**: Reads and displays all saved configuration profiles. If a file path is broken, the built-in error handling skips it gracefully without crashing the whole application.
- **Smart Path Parser**: Automatically strips accidental quotation marks and double extensions from user input.
